# 微信小程序运行指南

## 问题说明
当前项目依赖安装遇到问题（puppeteer模块被锁定），导致无法正常编译。

## 解决方案

### 方案1：使用HBuilderX（推荐）

1. **下载并安装HBuilderX**
   - 访问：https://www.dcloud.io/hbuilderx.html
   - 下载标准版或App开发版

2. **打开项目**
   - 启动HBuilderX
   - 文件 -> 打开目录
   - 选择 `D:\小程序开发练习\mini-shop-frontend`

3. **配置微信小程序AppID**
   - 在HBuilderX中打开 `manifest.json`
   - 找到"微信小程序配置"
   - 填写你的微信小程序AppID（如果没有，可以暂时使用测试号）

4. **运行到微信开发者工具**
   - 点击菜单：运行 -> 运行到小程序模拟器 -> 微信开发者工具
   - HBuilderX会自动编译并打开微信开发者工具

5. **在微信开发者工具中预览**
   - HBuilderX会自动将编译后的文件发送到微信开发者工具
   - 如果没有自动打开，手动打开微信开发者工具
   - 导入项目，选择目录：`D:\小程序开发练习\mini-shop-frontend\unpackage\dist\dev\mp-weixin`

### 方案2：手动修复依赖后编译

如果必须使用命令行，请按以下步骤操作：

1. **关闭所有可能占用文件的进程**
   ```powershell
   # 关闭微信开发者工具
   # 关闭所有Node.js进程
   taskkill /F /IM node.exe
   ```

2. **完全清理并重新安装**
   ```powershell
   cd "D:\小程序开发练习\mini-shop-frontend"
   
   # 强制删除node_modules（可能需要重启电脑后才能完全删除）
   Remove-Item -Recurse -Force node_modules -ErrorAction SilentlyContinue
   Remove-Item package-lock.json -ErrorAction SilentlyContinue
   
   # 使用淘宝镜像
   npm config set registry https://registry.npmmirror.com
   
   # 重新安装
   npm install
   ```

3. **编译微信小程序**
   ```powershell
   npm run dev:mp-weixin
   ```

4. **用微信开发者工具打开**
   - 打开微信开发者工具
   - 导入项目
   - 项目目录：`D:\小程序开发练习\mini-shop-frontend\unpackage\dist\dev\mp-weixin`
   - AppID：填写你的小程序AppID或使用测试号

### 方案3：创建新的uni-app项目并迁移代码

1. **使用HBuilderX创建新的uni-app项目**
   - 文件 -> 新建 -> 项目
   - 选择uni-app模板
   - 选择Vue2版本

2. **复制源代码**
   - 将原项目的以下文件夹复制到新项目：
     - `pages/`
     - `components/`
     - `api/`
     - `store/`
     - `static/`
   - 复制配置文件：
     - `pages.json`
     - `manifest.json`（需要重新配置）
     - `App.vue`
     - `main.js`
     - `uni.scss`

3. **重新安装依赖并运行**
   - 在新项目中右键 -> 外部命令 -> 终端
   - 执行 `npm install`
   - 运行到微信开发者工具

## 后端服务配置

无论使用哪种方案，都需要确保后端服务正在运行：

1. **启动后端**
   ```powershell
   cd "D:\小程序开发练习\mini-shop-backend"
   .\mvnw.cmd spring-boot:run
   ```

2. **确认后端地址**
   - 默认地址：http://localhost:8090
   - 检查 `mini-shop-frontend/api/request.js` 中的 `BASE_URL` 配置

3. **微信开发者工具配置**
   - 打开微信开发者工具
   - 详情 -> 本地设置
   - 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
   - 这样才能访问本地的 http://localhost:8090

## 常见问题

### 1. 依赖安装失败
**原因**：puppeteer等模块被锁定
**解决**：
- 重启电脑后再试
- 或者使用HBuilderX（方案1）

### 2. 微信开发者工具无法连接后端
**解决**：
- 确认后端已启动
- 在微信开发者工具中勾选"不校验合法域名"
- 检查防火墙设置

### 3. 编译后文件不存在
**解决**：
- 确保编译命令执行成功
- 检查 `unpackage/dist/dev/mp-weixin` 目录是否存在
- 如果使用HBuilderX，等待编译完成

### 4. Node.js版本问题
当前使用的是Node.js v24.15.0，某些旧版本的uni-app插件可能不兼容。
**建议**：使用Node.js 16或18版本

## 快速开始（最简单的方法）

1. 安装HBuilderX
2. 打开 `mini-shop-frontend` 项目
3. 点击"运行 -> 运行到小程序模拟器 -> 微信开发者工具"
4. 在微信开发者工具中勾选"不校验合法域名"
5. 启动后端服务
6. 完成！
