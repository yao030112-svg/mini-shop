# 实施计划：淘宝类型购物小程序

## 概述

本计划按照"先后端再前端、先基础设施再业务功能"的顺序组织任务。后端使用 Spring Boot + MyBatis + MySQL，前端使用 uni-app + Vue。所有代码位于 `d:\小程序开发练习` 工作区下。
