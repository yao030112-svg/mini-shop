# 快速启动指南 - 在微信开发者工具中运行

## 🚀 最快速的启动方法（推荐）

### 第一步：安装HBuilderX

1. 访问 https://www.dcloud.io/hbuilderx.html
2. 下载并安装 HBuilderX（标准版即可）

### 第二步：用HBuilderX打开项目

1. 启动 HBuilderX
2. 点击菜单：`文件` -> `打开目录`
3. 选择文件夹：`D:\小程序开发练习\mini-shop-frontend`

### 第三步：配置微信小程序

1. 在HBuilderX中双击打开 `manifest.json`
2. 找到左侧的"微信小程序配置"
3. 填写微信小程序AppID：
   - 如果没有AppID，可以注册一个测试号
   - 或者暂时留空，使用默认配置

### 第四步：运行到微信开发者工具

1. 在HBuilderX中点击菜单：`运行` -> `运行到小程序模拟器` -> `微信开发者工具`
2. 等待编译完成（底部控制台会显示进度）
3. HBuilderX会自动打开微信开发者工具并加载项目

### 第五步：配置微信开发者工具

1. 在微信开发者工具中，点击右上角的`详情`
2. 切换到`本地设置`标签
3. ✅ 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"

### 第六步：启动后端服务

打开一个新的PowerShell窗口：
```powershell
cd "D:\小程序开发练习\mini-shop-backend"
.\mvnw.cmd spring-boot:run
```

等待后端启动完成（看到"Started MiniShopApplication"字样）

### ✅ 完成！

现在你应该能在微信开发者工具中看到小程序界面了！

## 🔧 备选方案：使用命令行（如果HBuilderX不可用）

### 步骤1：重启电脑（重要！）

由于node_modules中的puppeteer模块被锁定，需要先重启电脑释放文件锁。

### 步骤2：运行安装脚本

重启后，右键点击以下文件并选择"使用PowerShell运行"：
```
D:\小程序开发练习\mini-shop-frontend\install-and-build.ps1
```

### 步骤3：用微信开发者工具打开编译后的项目

1. 打开微信开发者工具
2. 点击"+"创建新项目
3. 项目目录：`D:\小程序开发练习\mini-shop-frontend\unpackage\dist\dev\mp-weixin`
4. AppID：填写你的小程序AppID或使用测试号
5. 点击"导入"

### 步骤4：配置和启动后端

同上（第六步）

## ⚙️ 重要配置检查

### 1. 后端API地址
确保 `mini-shop-frontend/api/request.js` 中的BASE_URL正确：
```javascript
const BASE_URL = 'http://localhost:8090'
```

### 2. 微信开发者工具设置
必须勾选"不校验合法域名"，否则无法访问本地后端

### 3. 端口检查
- 后端运行在：8090端口
- 确保没有其他程序占用此端口

## ❓ 常见问题排查

### 问题1：微信开发者工具显示空白页
**解决方案：**
- 检查后端是否已启动
- 检查微信开发者工具是否勾选了"不校验合法域名"
- 查看控制台是否有错误信息

### 问题2：网络请求失败
**解决方案：**
- 确认后端服务正在运行
- 访问 http://localhost:8090 测试后端是否正常
- 检查防火墙设置

### 问题3：编译失败
**解决方案：**
- 使用HBuilderX方式（推荐）
- 或者重启电脑后运行 install-and-build.ps1 脚本

### 问题4：样式显示异常
**解决方案：**
- 等待编译完全完成
- 重新编译一次
- 检查 uni.scss 文件是否正确导入
